Introduction.

Bearhtml est un visualiseur de pages HTML, ce n'est pas un navigateur. L'impossibilité d'afficher une page peut provenir d'une des raisons suivantes :
  1. Mauvais codage Html
  2. Codage dynamique (script)
Dans ces cas vous pouvez essayer de visualiser la page en choississant "Visualiser dans le navigateur" avec le clic droit

Graphiques

Les URL graphiques distantes sont affichées . Pour voir les graphiques distants, cliquez sur Shift + Ctrl + 'U ', ou faites un clic droit dans la zone du message (mais pas sur un graphique) et choisissez "Montrer les images", ou assurez-vous que l'option Lecteur de Message est positionnée pour aller les chercher automatiquement. Il y aura un délai d'attente pendant la recherche des images. Pendant le chargement des images, un compteur montre la progression. Ne fermez pas le message avant que le compteur n'ait terminé le compte à rebours.
Dans le cas de fichiers images corrompus ou tronqués une petite fenêtre contenant un point d'exclamation s'affichera pour indiquer la présence d'une image non affichable. Les fichiers images invalides resteront sous cette forme : 

Html invalide

Si Bearhtml est incapable d'afficher un message en HTML cela entraîne un appel à LibTidy pour tenter une réparation. Une tentative sera faite pour afficher les images réparées. Si cela échoue le source html d'origine sera affiché. Une erreur à ce moment là indique un message html de particulièrement mauvaise qualité. Si l'affichage est vide ou de mauvaise qualité, choisir "Ouvrir dans le navigateur" avec le clic droit de la souris.
Vous voulez tenter le dépannage par vous-même ? Tout en gardant le message ouvert, lancez l'Explorateur Windows et rendez-vous dans votre dossier de fichiers temporaires (soit C:\Temp, C:\Windows\Temp ou C:\Documents and settings\nom_du_profil\Temp) et cherchez un fichier ayant un nom du genre WPM$1234.PM$ qui vient d'être créé. Ouvrez-le avec le Bloc-Notes, il devrait contenir du code HTML. Sauvegardez-le en tant que fichier HTML, par exemple wpm$1234.htm puis ouvrez-le dans votre navigateur web.

Jeux de caractères

Le code HTML est affiché en utilisant le jeu de caractère normal (c'est à dire ISO-8859-1). Si vous voyez des carrés vides là où le texte devrait être affiché, c'est l'indication que l'expéditeur utilise un jeu de caractères différent. Utilisez la même méthode de dépannage que ci-dessus, mais une fois dans votre navigateur allez dans le menu Encodage des caractères (ou toute autre appellation, l'emplacement dépend du navigateur utilisé) et sélectionnez une des méthodes proposées dans la liste. Si l'une de ces méthodes permet de lire le message, vous devriez informer l'expéditeur du problème de jeu de caractères et lui indiquer le jeu convenable.
Windows 95 possède une gestion des jeux de caractères limitée, pour corriger ce problème placez le paramètre CheckUTF8 à false (CheckUTF8=false) dans le fichier Bearhtml.ini.

Html suspect

La dangerosité des URL sur lesquelles vous cliquez est vérifiée, par exemple les URL dangereuses qui vous demandent des informations personnelles, soit :
  • Vérification des adresses IP numériques comme http://191.168.0.1
  • Vérification des liens vers 2 domaines, comme http://www.bonsite.com@www.mauvaissite.com
  • Vérification des liens trompeurs, c'est à dire quand le lien affiché est différent du lien cible
  • Vérification des noms de domaines masqués comme http://%37icrosoft.com
  • Vérification des URL dont le contenu fichier est présent dans Bearwarn.txt (tests d'hameçonnage)
  • Vérification de l'appartenance de l'URL à la liste noire du service surbl.org
Si un problème est détecté un message vous en avertira. Cliquer sur OK lancera le navigateur. En cliquant sur Annuler l'URL sera mémorisée pour le reste de la session de Pegasus Mail. Le passage du curseur de la souris au-dessus de l'URL le transformera en panneau Stop.
La vérification de la liste noire se fait ou non selon la valeur de WantBl (Yes/No) dans le fichier BearHtml.ini. Cliquez sur le lien, la ligne d'état affichera le résultat.

Détection de tentative d'hameçonnage

En plus de la vérification de la liste noire, Bearhtml examine également les hyper liens affichés. Si des noms de domaines connus appartenant à de entreprises commerciales apparaissent ailleurs que dans la partie correcte de l'URL, un message d'avertissement est affiché. Les utilisateurs peuvent compléter la liste contenue dans le fichier Bearwarn.txt avec leurs propres noms de domaine. Voir le fichier Bearwarn.txt pour plus d'informations sur ce sujet.

Scripting

Aucune forme de script tel que le Javascript n'est supportée. Tout codage de ce type dans un message est ignoré ce qui rend l'affichage de tels messages problèmatique. En cas de besoin le code HTML doit être transmis à un navigateur.
De plus toutes les demandes d'écriture ou de lecture de Cookies sont ignorées.

Impression

Pour l'instant Bearhtml ne peut pas imprimer les messages avec l'information d'enveloppe, il peut simplement imprimer la page HTML affichée sur l'imprimante Windows par défaut. Optionally the printer can be changed by setting the PrinterSetup= value to Yes in Bearhtml.ini.

Styles

Les définitions de styles intégrées sont traitées. Les styles situés sur des URL distantes ne sont pas chargés.

Taille de police

L'augmentation ou la diminution de la taille de la police d'affichage en utilisant la touche Ctrl associée à la roulette de la souris peut ne pas fonctionner à tous les coups, et peut causer dans certains cas un affichage surprenant. Si un message possède déjà des tailles de police intégrées elles sont prioritaires sur des tailles par défaut. Certains messages ayant certaines tailles de police intégrées et d'autres laissées par défaut  peuvent apparaître  comme pauvrement formatés. Veuillez noter qu'il existe de multiples méthodes pour ajuster la taille des polices à l'aide du clavier.

Journalisation

Pour les problèmes de maintenance un fichier journal peut être créé dans le dossier de nouveau courrier de Pegasus Mail (par ex. c:\pmail\mail), c'est un fichier texte portant le nom de Bearhtml.log qui peut être consulté à l'aide du Bloc-Notes.
Dans le cas où il est fait appel à LibTidy pour tenter de réparer du html, un fichier Beartidy.txt contenant le compte rendu de Tidy pourra être trouvé dans le répertoire de nouveau courrier de l'utilisateur.

Délais

Bearhtml utilise les services Internet de Windows (WinInet.dll) pour exécuter ses recherches DNS et télécharger les pages web. Occasionnellement la connexion Internet peut ralentir ou les services DNS peuvent d'arrêter. Un délai de connexion est détecté par les services Internet Windows et Bearhtml en est informé (la valeur par défaut est de 60 secondes). La valeur de MaxFetchTime dans Bearhtml indique le temps maximum d'attente, il doit être inférieur au délai Windows. Si le temps de téléchargement dépasse la valeur de MaxFetchTime tous les téléchargements suivants pour ce message seront arrêtés.

Déplacement dans la page

Il existe deux façons de se déplacer vers le haut ou vers le bas dans une page web. Les ascenseurs sur le coté de la fenêtre et les touches directionnelles du clavier. Les ascenseurs utilisent toujours la fonction de Windows de déplacement d'un certain nombre de pixels dans une direction ou dans l'autre, sans prendre en compte la position du curseur.
L'utilisation du clavier permet de se déplacer par lignes tout en repositionnant le curseur texte. Dans ce cas la valeur du déplacement peut être facilement prévisible, mais dans le cas où du texte et des images sont mélangés, la valeur du déplacement peut être difficile à prévoir. Bearhtml essayera, par exemple, de déplacer le curseur vers le haut ou vers le bas d'un nombre de pixels correspondant à une hauteur de page et essayera de trouver une ligne de texte totalement affichée pour y placer le curseur.

Redirection

Le protocole Http permet qu'une requête vers une Url soit modifée par le serveur web pour aller chercher un objet (page ou image) à un autre emplacement. Bearhtml peut accepter cette fonctionnalité mais par défaut  elle est désactivée. Il faut prendre garde à activer la redirection. La description des risques encourus n'est pas l'objet de ce fichier. Faites une recherche sur Google telle que : http://computing-dictionary.thefreedictionary.com/Http+redirection

Installation

  1. Fermez Pegasus Mail
  2. Décompressez le contenu de l'archive bearhtml.zip dans le répertoire de l'exécutable Pegasus Mail (par exemple c:\pmail).
  3. Si nécessaire, éditez Bearhtml.ini et Bearwarn.txt
  4. Si nécessaire, sauvegardez Bearhtml.ini dans le répertoire de nouveau courrier de l'utilisateur pour écraser les valeurs d'origine
  5. Relancez Pegasus Mail

Désinstallation

  1. Fermez Pegasus Mail
  2. Renommez ou suprimez Bearhtml.dll
  3. Relancez Pegasus Mail

Contenu de Bearhtml.ini
  • ScrollPixels= (défaut 20) nombre de pixels de décalage avec la roulette de la souris
  • ScrollRepeat= (défaut 1) répétition du décalage de la roulette de la souris
  • ScrollDelayMs= (défaut 1 ms) delai entre 2 actions sur la roulette de la souris
  • DefaultFontSize= (défaut 12) Taille d'affichage de la police par défaut
  • DefaultFontName= (défaut Arial) Police par défaut
  • WantBl= (défaut No, Yes/No) Vérification de la liste noire
  • WantCache= (défaut Yes, Yes/No) Sauvegarde les images dans le sous-répertoire RmtFiles du répertoire de courrier
  • LocalIPAddress= (défaut No) utilisé pour éviter que les adresses locales du type 127.0.0.1 ne causent des fausses alertes d'hameçonnage, plusieurs adresses séparées par des virgules. Par exemple localIPAddress=127.0.0,192.168 exclut les plages d'adresses 127.0.0.x et 192.168.y.z.
  • MaxFetchTime= (défaut 25) temps maximum acceptable de téléchargement, en secondes (voir ci-dessus : Délais)
  • CheckUTF8= (défaut false) Recherche de l'en-tête du jeu de caractères MIME UTF8
  • Redirection= (défaut false) redirection Http vers une Url située à un autre emplacement (risqué)
  • PrinterSetup= (défaut No) Permet la sélection de l'imprimante lors de l'impression d'une page html
  • Loglevel= (défaut 3) Utilise le même degré de journalisation que Pegasus Mail, soit :
    Loglevel=1 Signale les erreurs critiques des fonctions de Bearhtml
    Loglevel=2 Idem Loglevel=1 ainsi que les erreurs sérieuses
    Loglevel=3 Idem Loglevel=1/2 ainsi que les événements normaux
    Loglevel=4 Idem Loglevel 1/2/3 ainsi que les informations détaillées de transactions

    Les utilisateurs de Windows 95 ont un jeu de caractères limité et doivent définir CheckUTF8=false dans Bearhtml.ini.
